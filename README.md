# Python Fibonacci Generator

Generates a Fibonacci sequence of any length — just pass a number and get back the series. Handy for math exercises, algorithm practice, or whenever you need a quick Fibonacci fix.

## Features

- **One function, zero fuss** — call `fibonacci(n)`, get your sequence
- **Type-safe** — raises `TypeError` on non-integer input (no surprises)
- **Edge-case aware** — handles 0, 1, and negative lengths gracefully
- **Clean & documented** — full type hints and a clear docstring
